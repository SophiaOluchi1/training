key takeaways

  • aria-label - label elements without visible text
  • aria-labelledby-use existing text as label
  • aria-describedby-connect help text to inputs
  • aria-hidden-hide decorative content from screen readers
  • aria-expanded-show expanded/collapsed state
  • aria-current-mark current page/item
  • aria-invalid-mark form errors
  • role="alert"-urgent announcement
  • role="status"-Non-urgent updates
  • scope-define table header relationships

golden rule:use semantic html first, add ARIA only when needed!
